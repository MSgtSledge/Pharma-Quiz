                    PHARMA-QUIZ — SAB ANÄSTHESIEPFLEGE

Interaktives Online-Quiz für die Pharmakologie-Prüfung der Spezialisierung
Anästhesiepflege (SAB) am AKH Wien.

================================================================================
WAS IST DAS?
================================================================================

Ein HTML5-basiertes Quiz mit 144 echten Pharmakologie-Fragen aus dem Stefan 
Bartos Vorlesungsskript (SAB Anästhesiepflege, AKH Wien).

FEATURES:
  ✓ 144 Fragen aus dem offiziellen Curriculum
    (Grundlagen, Anästhetika, Opioide, Muskelrelaxanzien, Lokalanästhetika,
     Hämodynamik, Infusionen, Alpha-2, Antiemetika, CPR, etc.)
  
  ✓ Zufällige Fragen — neue Reihenfolge bei jedem Start
  
  ✓ Zufällige Antworten — Optionen gemischt, damit nicht immer die gleiche 
    Position richtig ist
  
  ✓ Sofortiges Feedback — richtig (grün) oder falsch (rot) nach jeder Antwort
  
  ✓ Fortschrittsanzeige — Sehe, wo du bist im Quiz
  
  ✓ Finale Auswertung — Dein Score am Ende
  
  ✓ Offline-ready — Funktioniert komplett im Browser, keine 
    Internetverbindung nötig nach dem ersten Laden

================================================================================
FRAGEN-KATEGORIEN
================================================================================

Die 144 Fragen decken ab:

  • Inhalationsanästhetika
    (MAC, Sevofluran, Desfluran, Lachgas)
  
  • Intravenöse Anästhetika
    (Propofol, Esketamin, Midazolam, Etomidat, Thiopental)
  
  • Opioide
    (Morphin, Fentanyl, Remifentanil, Piritramid, Nalbuphin, Naloxon)
  
  • Muskelrelaxanzien
    (Succinylcholin, Rocuronium, Cis-Atracurium, Sugammadex, Neostigmin)
  
  • Lokalanästhetika
    (Estertyp, Amidtyp, LAST, Lipid-Rescue)
  
  • Hämodynamik
    (Adrenalin, Noradrenalin, Phenylephrin, Esmolol, Dobutamin)
  
  • Infusionen
    (Volumen, Elektrolyte, Blutprodukte)
  
  • Alpha-2-Agonisten
    (Clonidin, Dexmedetomidin)
  
  • Antiemetika
    (Ondansetron, Droperidol, Dexamethason, PONV-Prophylaxe)
  
  • NSAR & Analgetika
    (Diclofenac, Paracetamol, Metamizol, Parecoxib)
  
  • Reanimation
    (ALS-Algorithmus, Dosierungen, Amiodaron)
